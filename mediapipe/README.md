## Objetivo

Aplicação de Human Pose Estimation utilizando MediaPipe Tasks para detectar 33 landmarks corporais em vídeos.

## Tecnologias

- Python
- MediaPipe
- OpenCV
- Google Colab

## Funcionalidades

- Detecção de pose humana
- Processamento de vídeos
- Geração de vídeo anotado
- Visualização dos landmarks e conexões corporais

## Como executar

1. Abra o notebook no Colab.
2. Execute todas as células.
3. Faça upload do vídeo.
4. Aguarde o processamento.
5. Baixe o vídeo resultante.
